# PLAN: Content Replacement — Vân Hoa Website

## Overview
Thay thế toàn bộ nội dung text trên website Vân Hoa dựa trên tài liệu kinh doanh "KHỞI SỰ KINH DOANH N5.docx", vẽ lại logo SVG theo mẫu image.png. **Giữ nguyên 100% design, layout, CSS, JS.**

## Project Type: WEB (Content Update)

## Success Criteria
- [ ] Tất cả nội dung text đã được cập nhật theo tài liệu
- [ ] Logo SVG mới theo phong cách line-art giống image.png
- [ ] Layout/design không thay đổi
- [ ] SEO meta tags cập nhật
- [ ] Website hoạt động bình thường

## Task Breakdown

### Task 1: Vẽ lại Logo SVG
- **Agent:** `frontend-specialist`
- **Priority:** P1
- **INPUT:** image.png (mẫu logo line-art bông hoa + calligraphy)
- **OUTPUT:** logo.svg mới với phong cách line-art
- **VERIFY:** Logo hiển thị đúng trên navbar và footer

### Task 2: Cập nhật SEO & Meta
- **Agent:** `frontend-specialist`  
- **Priority:** P2
- **INPUT:** Thông tin thương hiệu từ docx
- **OUTPUT:** title, description, keywords, OG tags, Schema.org cập nhật
- **VERIFY:** View source kiểm tra meta tags

### Task 3: Cập nhật Hero Section
- **Agent:** `frontend-specialist`
- **Priority:** P2
- **INPUT:** Brand story + product info từ docx
- **OUTPUT:** Label, subtitle, description mới
- **VERIFY:** Hiển thị đúng trên trang

### Task 4: Cập nhật Brand Story Section
- **Agent:** `frontend-specialist`
- **Priority:** P2
- **INPUT:** Câu chuyện thương hiệu từ docx
- **OUTPUT:** Quote, paragraphs mới
- **VERIFY:** Nội dung khớp tài liệu gốc

### Task 5: Cập nhật Collections & Products
- **Agent:** `frontend-specialist`
- **Priority:** P2
- **INPUT:** Thông tin BST + sản phẩm từ docx
- **OUTPUT:** Mô tả BST và sản phẩm cập nhật
- **VERIFY:** Text mới hiển thị đúng

### Task 6: Cập nhật Why Us & Contact & Footer
- **Agent:** `frontend-specialist`
- **Priority:** P3
- **INPUT:** Điểm khác biệt + kênh phân phối từ docx
- **OUTPUT:** Feature cards, contact text, footer tagline mới
- **VERIFY:** Tất cả sections hiển thị đúng

## Phase X: Verification
- [ ] Không có thay đổi CSS/JS
- [ ] Tất cả links nội bộ hoạt động
- [ ] Logo mới render đúng
- [ ] Responsive layout giữ nguyên
- [ ] Text content khớp tài liệu kinh doanh
